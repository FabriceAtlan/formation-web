Cours sur la méthodologie BEM CSS

1. Pourquoi utiliser BEM ?

BEM (Block, Element, Modifier) facilite la gestion du CSS pour des projets de grande envergure en apportant une structure claire.

  • Clarté : Les noms de classes décrivent leur rôle dans l'interface.
  • Modularité : Les composants sont facilement réutilisables.
  • Scalabilité : Le CSS reste maintenable même pour des interfaces complexes.

2. Les principes de BEM

La méthodologie BEM repose sur trois concepts principaux :

  • Block (Bloc) : Un composant indépendant de l'interface.
  • Element (Élément) : Une partie d'un bloc qui a une fonction spécifique.
  • Modifier (Modificateur) : Une variation d'un bloc ou d'un élément.

3. Syntaxe de base

Voici la structure de base de BEM :

.block {} .block__element {} .block--modifier {} .block__element--modifier {}

Exemple d'un bouton avec un icône :

<button class="button button--primary"> <span class="button__icon"></span> <span class="button__text">Click Me</span> </button>

4. Exemple détaillé

Exemple complet d'une carte avec HTML et CSS :

HTML :

<div class="card card--featured"> <h2 class="card__title">Titre de la carte</h2> <p class="card__description">Description de la carte.</p> <button class="card__button card__button--large">En savoir plus</button> </div>

CSS :

/* Block */ .card { background-color: #fff; border: 1px solid #ccc; padding: 20px; } /* Element */ .card__title { font-size: 1.5rem; color: #333; } .card__description { font-size: 1rem; color: #666; } /* Modifier */ .card--featured { background-color: #f9f9f9; border-color: #ff5722; } .card__button { padding: 10px 15px; background-color: #2196f3; color: white; } .card__button--large { padding: 15px 25px; }

5. BEM et les préprocesseurs CSS

BEM s'intègre bien avec les préprocesseurs CSS comme Sass. Exemple avec Sass :

.card { background-color: #fff; border: 1px solid #ccc; padding: 20px; &__title { font-size: 1.5rem; color: #333; } &__description { font-size: 1rem; color: #666; } &--featured { background-color: #f9f9f9; border-color: #ff5722; } &__button { padding: 10px 15px; background-color: #2196f3; color: white; &--large { padding: 15px 25px; } } }

6. Conclusion

BEM est une méthode efficace pour organiser votre CSS en blocs, éléments et modificateurs, rendant le code plus clair et plus facile à maintenir.

  • Block : Le composant principal.
  • Element : Une partie spécifique du composant.
  • Modifier : Une variation du composant ou d’un élément.
