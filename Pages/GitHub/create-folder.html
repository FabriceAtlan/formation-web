Créer un Dossier sur GitHub Mobile

Suivez ces étapes simples pour créer un dossier dans un dépôt GitHub à partir de la version mobile :

  • 1. Ouvrez votre dépôt GitHub depuis l'application mobile ou le site mobile.
  • 2. Cliquez sur l'icône "Ajouter un fichier" (ou similaire selon l'interface).
  • 3. Nommez votre fichier en incluant le chemin du dossier, par exemple : mon_dossier/mon_fichier.txt.
  • 4. Ajoutez du contenu dans le fichier (vous pouvez également laisser le fichier vide).
  • 5. Confirmez la création du fichier en le commitant dans votre dépôt.

GitHub créera automatiquement le dossier si celui-ci n'existe pas encore. Cela permet de contourner la limitation qui empêche de créer des dossiers vides.
