Création d'un dossier sur mobile et PC

GitHub ne permet pas de créer un dossier vide, vous devez donc créer au moins un fichier dans le dossier. Suivre les étapes suivantes pour créer un dossier dans un dépôt GitHub :

  • 1. Ouvrir le dépôt GitHub.
  • 2. Cliquez sur l'icône Ajouter un fichier.
  • 3. Nommez le fichier en incluant le chemin du dossier, par exemple : mon_dossier/mon_fichier.txt.
  • 4. Ajoutez du contenu dans le fichier (vous pouvez également laisser le fichier vide).
  • 5. Confirmez la création du fichier par un commit.

GitHub créera automatiquement le dossier si celui-ci n'existe pas encore. Cela permet de contourner la limitation qui empêche de créer des dossiers vides.
